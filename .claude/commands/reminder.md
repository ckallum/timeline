Read the `reminder` skill. Then parse the user's reminder.

Usage:
- `/reminder [text with date]` — set a reminder
- `/reminder [text]` — if no date is found, ask for one

Examples:
- `/reminder call mum tomorrow`
- `/reminder dentist appointment 2026-05-01`
- `remind me to review the PR before Friday`
